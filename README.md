MoonPlayer
=============

MoonPlayer一款基于FFmpeg + OpenGL + Qt 打造的视频播放器

## 3rdpart
第三方依赖库
* `ffmpeg` 音视频编解码库.

## MoonPlayer
基于C++ 11实现的跨平台播放器核心逻辑
* `MoonAV` 基于ffmpeg封装的音视频解码类相关
* `MoonPlayer` 播放器核心逻辑实现
* `MoonThread` Thread封装的线程类
* `MoonQueue` 线程安全队列

## MoonPlayerClient

基于QT的客户端界面实现
